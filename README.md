# StudentManagementSystem
This is a web-based Student Management System developed using Java, Servlets, JDBC, and Oracle Database.
It helps manage student details, marks, and generate reports efficiently.

#features
Add Student Details
View All Students
Add Marks with Exam Type (Semester-wise)
Update Marks
Generate Student Report by Roll Number
Compare Marks (1st Sem, 2nd Sem, etc.)
Calculate Total & Percentage

#Technologies Used
Java (Core Java)
Servlets (Jakarta)
JDBC (Database Connectivity)
Oracle SQL
HTML, CSS (Frontend)
Apache Tomcat (Server)

#Concepts Used
DAO (Data Access Object) Design Pattern
Database Operations (CRUD)
